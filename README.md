# covid19-kr
KR COVID-19 Informative Repository (Kangwon National University, CSE)

# 개요
- Node.js에서 손쉽게 사용가능한 COVID-19(코로나바이러스-19) API 패키지
- 강원대학교 SW학습공동체 오픈소스 프로젝트 팀 **Bear Soup**
